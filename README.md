# Dog Quiz - Adivinhe a Raça

Um jogo mobile minimalista feito com React Native e Expo,
onde o objetivo é adivinhar a raça do cachorro exibido na tela.

## 🚀 Como rodar o projeto

### Pré-requisitos

*   Node.js instalado
*   Gerenciador de pacotes (npm, pnpm, bun...)
*   Aplicativo **Expo Go** instalado no seu celular (disponível na App Store ou Google Play)

### Passo a passo

1.  **Instale as dependências:**
    ```bash
    npm install
    ```

2.  **Inicie o servidor do Expo:**
    ```bash
    npm run start
    ```

3.  **Abra o jogo:**
    *   Escaneie o QR Code que aparecerá no terminal usando a câmera do seu celular (iOS) ou o app do Expo Go (Android).
    *   O jogo será carregado automaticamente no seu dispositivo.

## 🛠️ Tecnologias

*   [React Native](https://reactnative.dev/)
*   [Expo Go](https://expo.dev/go)
*   [Dog API](https://dog.ceo/dog-api/) (Fonte dos dados e imagens)
